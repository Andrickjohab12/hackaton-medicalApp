<h2>Medical App</h2>
<br><br>
Interfaz gráfica de una pagina para el diagnostico médico. Realizada durante el Hackaton del departamento de sistemas computacionales 2025 en el Instituto tecnologico de méxico (ITT)
<br><br>
Desarrolada con React,Next.js y Typescript
Para correr el servidor necesitarras los comandos:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Abre [http://localhost:3000](http://localhost:3000) con tu navegador para ver el resultado.
